# 🌐 PortfolioMern

![Portfolio Screenshot](https://portfolio-mern-sage.vercel.app/preview.png)

🚀 A modern and fully responsive portfolio website built with the MERN stack, showcasing projects, skills, and an API-based contact form. Hosted on **Vercel**.

## 🔗 Live Demo
👉 [Visit Portfolio](https://portfolio-mern-sage.vercel.app/)

---

## 📌 Features

✅ **Modern Glassmorphism UI** – A sleek, modern interface with smooth animations and intuitive navigation.
✅ **Full Responsiveness** – Optimized for all devices, from desktops to mobile.
✅ **Dynamic Project Showcase** – Easily update and display projects dynamically.
✅ **API-Based Contact Form** – Secure and real-time email communication.
✅ **Fast & Optimized** – Deployed on Vercel for high performance.

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT & Cookies
- **Deployment:** Vercel (Frontend & Backend)

---

## 🏗️ Installation & Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/ABDLSamaD/PortfolioMern.git
   cd PortfolioMern
   ```
2. **Install Dependencies**
   ```bash
   npm install
   ```
3. **Start the Development Server**
   ```bash
   npm run dev
   ```
4. **Environment Variables** (Create a `.env` file in the root directory and add the necessary keys)
